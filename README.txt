== Description ==

Users can easily add TinyMCE Editor in Comment Form in just one click.
User can also disable heading tags,Media Button & Pre Tags in tinymce editor. (Visual Mode)

== Installation ==

1. Upload `comments-tinymce` plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. You will find Comment Form Editor with TinyMCE Plugin in your WordPress admin screen.
